# 🗄️ Aloisio Ramon

Me chamo **Aloisio Ramon Araujo de Almeida**, sou **DBA Júnior** com foco em **administração e sustentação de ambientes críticos Oracle e Microsoft SQL Server**, atuando com infraestrutura, observabilidade, automação e segurança de dados.

Atualmente trabalho na **WiseDB**, onde evoluí do estágio para a posição júnior, administrando bancos de dados de clientes de diversos segmentos (saúde, ERP, logística), com atuação prática em diagnóstico e resolução de incidentes, monitoramento proativo, rotinas de backup e restore e governança de acessos.

Sou **OCI Observability Professional certificado** e uso **IA no meu dia a dia de trabalho**: desenvolvi skills personalizadas no Claude para gerar runbooks padronizados, registrar o passo a passo de cada execução e manter a auditoria interna organizada. Automação com IA pra mim não é teoria, é rotina.

Curso **Sistemas de Informação** e compartilho conhecimento técnico sobre monitoramento de banco de dados no meu canal do YouTube.
